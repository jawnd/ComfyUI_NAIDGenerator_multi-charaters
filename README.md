# comfyui_naidgenerator_multi-charaters

修改了原版naidgenerator的部分功能，加了一些节点

* 增加角色提示词节点，可以像novelai网页版一样自定义添加多个角色并自定义这些角色的位置
* 加入随机图片尺寸节点
* 加入随机延时节点
* 删除自动保存

更多说明详见演示工作流

原版：https://github.com/bedovyy/ComfyUI_NAIDGenerator

# comfyui_naidgenerator_multi-charaters

Modified some features of the original naidgenerator and added a few nodes

* Added a character prompt node, allowing you to custom add multiple characters and customize their positions just like the NovelAI web version
* Added a random image resolution node
* Added a random delay node
* Removed auto-save

For more details, please refer to the demo workflow
